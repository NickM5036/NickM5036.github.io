Dice Game

This Java code represents a text-based dice game where the player can choose between two actions: moving one space or rolling two dice to determine the number of spaces to move. The game continues until the player either exits voluntarily or reaches position 100, signifying a win. The code also includes methods to simulate dice rolls and calculate movement based on specific rules, ensuring that the player's progress aligns with the game's objectives.
